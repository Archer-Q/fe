# Maximum Depth of Binary Tree

## JavaScript
```javascript
function maxDepth(root) {
  return root ? 1 + Math.max(maxDepth(root.left), maxDepth(root.right)) : 0
}
```
## Ruby
```ruby
def max_depth(root)
  root.nil? ? 0 : 1 + [max_depth(root.left), max_depth(root.right)].max
end
```
