# Triangle

## JavaScript



`超时解法`
```javascript
function minimumTotal(triangle) {
  return min(triangle, 0, 0)
  function min(t, i, j) {
    return t[i + 1] ? t[i][j] + Math.min(min(t, i + 1, j), min(t, i + 1, j + 1)) : t[i][j]
  }
}
```
