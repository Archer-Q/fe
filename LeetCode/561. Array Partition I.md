# Array Partition I
`简单`

## JavaScript
```javascript
function arrayPairSum(nums) {
  return nums.sort((a, b) => a - b).reduce((sum, cur, idx) => {
    if (idx % 2 === 0) sum += cur
    return sum
  })
}
```

## Python
```python
class Solution(object):
    def arrayPairSum(self, nums):
        return sum(sorted(nums)[::2])
```
