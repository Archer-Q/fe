# AJAX 进阶


## GET

| 方法   | URL                                      | 作用                               |
|-------|------------------------------------------|------------------------------------|
| `GET` | `https://fe13.now.sh/api/posts`          | 获取 `Post` 列表                    |
| `GET` | `https://fe13.now.sh/api/posts/1`        | 获取 `id` 为 `1` 的 `Post`          |
| `GET` | `https://fe13.now.sh/api/posts?type=news`| 获取 `type` 为 `news` 的 `Post` 列表 |
