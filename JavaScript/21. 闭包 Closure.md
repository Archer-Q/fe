# 闭包 Closure

> A closure is the combination of a function and the lexical environment within which that function was declared.

闭包的两个要素
* 函数
* 函数声明时所处的词法环境


```javascript
let name = "Tom"

function greet() {
  console.log(`Hi, ${name}!`)
}

name = "Sam"
```

→
